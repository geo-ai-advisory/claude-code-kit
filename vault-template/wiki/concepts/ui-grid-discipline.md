# Вёрстка — ровная сетка, элементы в одну строку

## Суть
- Все контролы одного уровня — в одной flex-строке с `align-items: center`.
- Второстепенные кнопки (сброс кеша, refresh) — иконочные inline, не полноразмерные.
- Не создавать sidebar / колонку для 1-2 кнопок — встраивать в строку.
- Карточки / контент — на всю ширину, без пустых зон справа.
- `flex-wrap: wrap` только для responsive, не по умолчанию.

## Сохранено из memory: feedback_layout.md

При вёрстке любых UI-элементов — всё должно быть в одной строке, ровная логичная сетка. Не выносить элементы под основную строку контролов. Не оставлять пустое пространство.

**Why:** Пользователь ожидает компактный UI по стандартам веба. Ломка на новые строки, пустые зоны справа, sidebar ради 1-2 кнопок — всё это вызывает резкую реакцию.

**How to apply:**
1. Все контролы одного уровня — в одной flex-строке с `align-items: center`
2. Второстепенные кнопки (сброс кеша и т.п.) — маленькая иконочная кнопка inline, не полноразмерная
3. Не создавать sidebar/колонку для 1-2 кнопок — встраивать в строку контролов
4. Карточки/контент — на всю ширину, без пустых зон справа
5. Проверять визуально перед отдачей — элементы не должны "уезжать" вниз
6. `flex-wrap: wrap` только для responsive, не по умолчанию
7. **НЕ пушить UI-изменения в гит пока пользователь не скажет** — показать скриншот, дождаться подтверждения
